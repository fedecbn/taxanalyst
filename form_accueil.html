Cette application web vous permettra de
- faire le suivi des taxons dans les différentes versions de TaxRef à partir de son code nom (CD_NOM).
- laire une première analyse de l'agrégation des catalogues territoriaux du réseau

Si vous souhaitez faire le suivi d'un CD NOM dans TAXREF,
  • sélectionnez l'onglet "Suivi Taxon TAXREF",
  • entrez le cd_nom du taxon que vous souhaitez suivre,
  • sélectionnez les paramètres à afficher,
  • et cliquez sur OK

    Si vous souhaitez rechercher des information issus du TAXA,
  • sélectionnez l'onglet "TAXA - Catalogues CBN",
  • entrez une partie du nom scientifique,
  • selectionnez le taxon voulu

    Accédez également à la liste des taxons qui n'ont pas été rattachés à TAXREF à travers l'onglet "Taxons non rattachés"
